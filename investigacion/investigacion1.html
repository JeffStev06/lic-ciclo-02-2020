Funciones Math

Para las siguientes funciones son parte de un objeto llamado Math, que permite hacer tareas matemáticas, para utilizarlas es necesario anteponer esta palabra y acceder a la función con un punto [Math.round(2.5)]

abs()

Retorna el valor absoluto del número dado.

Uso: Math.round(2.5)

Ejemplo:

round()

Redondea el número dado al entero más cercano.

Uso: Math.round(2.5)

Ejemplo:

ceil()

Redondea el número dado al entero superior

Uso: Math.ceil(2.5)

Ejemplo:

floor()

Redondea el número dado al entero inferior

Uso: Math.floor(2.5)

Ejemplo:

exp()

Retorna ex donde x es el parámetro que pasamos a la función.

Uso: Math.exp(1)

Ejemplo:

log()

Retorna la funcion logarítmica natural de un número.

Uso: Math.log(1)

random()

Retorna un número aleatorio entre 0 y 1, Podemos hacer uso de otra sintaxis paraa obtener aleatorios de más cifras Así como en el ejemplo, es necesario usar una funcion de redondeo como las vistas anteriormente, para obtener valores enteros. Uso:

Math.random()
(Math.random() * 10) + 1
(Math.random() * 100) + 1
