MM161925 & MA161916
Null

Null se refiere a un objeto inexistente, que básicamente significa 'vacío' o 'nada'.

Undefined

El valor 'Undefined' denota que se ha declarado una variable, pero no se le ha asignado ningún valor. Entonces, el valor de la variable es 'indefinido'.

Recomendaciones

Se puede asignar manualmente el valor 'indefinido' a una variable, pero eso no es recomendable. Entonces, 'nulo' se asigna a una variable para especificar que la variable no contiene ningún valor o está vacía. Pero 'undefined' se usa para verificar si a la variable se le ha asignado algún valor después de la declaración.
